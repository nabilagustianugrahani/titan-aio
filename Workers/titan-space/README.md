# TITAN AIO Creator

Scene Builder + AI Voice + AI Music — totally free, no GPU needed.

## Features
- **Scene Builder**: Compose video clips, add text overlays, arrange timeline
- **Voice Generator**: Text-to-speech in Indonesian/English (edge-tts)
- **Music Generator**: AI background music from text prompt (MusicGen)
